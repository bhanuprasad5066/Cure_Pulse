FILE DESCRIPTIONS
=================

BACKEND FILES:
--------------
app.py              - Main application with routes, database queries, file uploads
config.py           - Database credentials, upload settings, Twilio config
schema.sql          - Creates database tables and inserts sample doctors

FRONTEND FILES:
---------------
templates/base.html     - Contains Tailwind CSS CDN, flash messages, header/footer

requirements.txt - Flask, flask-mysqldb, twilio, werkzeug
